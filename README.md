# global_earthquakes_10yrs_M4.5_2025-12-10
# Проект: анализ землетрясений (10 лет, M4.5+)
# 1. Резюме проекта
Данный проект был создан для анализа мировой сейсмической активности за последние 10 лет на основе датасета с землетрясениями магнитудой M4.5+.
В рамках проекта исследуются магнитуды, глубины, географическое распределение эпицентров, динамика по годам и ключевые сейсмоопасные регионы мира.

Цель анализа — выявить закономерности, определить наиболее активные зоны землетрясений и сформировать выводы, основанные на визуализации и статистической обработке данных.
Данные были предоставлены для сдачи экзамена в AlmaU.
# 2. Бизнес-вопросы
1. В каких регионах Земли землетрясения происходят чаще всего?
2. Как распределяется магнитуда землетрясений? Какие магнитуды встречаются чаще?
3. Есть ли сезонность или месячные пики землетрясений?
4. Как связаны глубина и магнитуда?
5. На какой глубине происходит больше всего землетрясений?
6. Какие страны или зоны разломов наиболее сейсмоопасные?
7. Как изменялась общая сейсмическая активность за последние 10 лет?
8. Какое количество сильных землетрясений (магнитуда ≥7) было по годам?
9. Какие регионы наиболее опасны по сочетанию глубины + магнитуды?
10. Есть ли корреляция между широтой, долготой и силой землетрясений?
